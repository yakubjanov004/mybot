Texnik xizmat

1. texnik bilan bajariladi ombor aralashuvisiz
client zayavka qoldiradi(zayavka haqida ma'lumotlarni stateda olgan bo'lamiz buni ham) -> zayavka controllerga keladi ->
Controller esa bu zayavkani texnikga biriktiradi -> texnik vazifani ko'radi -> manzilga boradi boshlash qiladi ->
-> diagnostika qo'yadi -> bot texnikadan ombor bilan ishlaydimi yokida yo'q shuni so'raydi -> bu holatda yo'qni bosadi(warehouse bilan zayavka to'ldirilmaydi) ->
-> muzmmoni tuzatib zayakaga qo'shimcha sifatida izoh qo'shadi shunda -> shundan so'ng zayavka yopiladi -> bu haqida clientga zayavka haqida ma'lumot yuboriladi ->
-> client esa zayavka to'g'riligini tasdiqaydi va 5a yulduzdan birini tanlaydi(sifatini baholash ma'nosida)

2. texnik bilan bajariladi ombor bilan ishlaydigan qismi
client zayavka qoldiradi(zayavka haqida ma'lumotlarni stateda olgan bo'lamiz buni ham) -> zayavka controllerga keladi ->
Controller esa bu zayavkani texnikga biriktiradi -> texnik vazifani ko'radi -> manzilga boradi boshlash qiladi ->
-> diagnostika qo'yadi -> bot texnikadan ombor bilan ishlaydimi yokida yo'q shuni so'raydi -> bu holatda xani bosadi(warehouse bilan zayavka ishlaydigan bo'ladi) ->
-> zayakaga qo'shimcha sifatida obordan nimalar ishlailishiga izoh qo'shadi va warehousega yuboriladi ->  zayavka warehousega keladi ->
-> warehouse esa buni ko'rib bazadan o'chirib qo'yadi va client uchun yozib qo'yadi -> warehouse qilgan ishi bo'yicha izox yozadi va texnikka yuboradi ->
-> texnik buni ko'radi va ishni yakunlagandan so'ng esa yakunlsh qiladi -> zayavka yopiladi ->  bu haqida clientga zayavka haqida ma'lumot yuboriladi ->
-> client esa zayavka to'g'riligini tasdiqaydi va 5a yulduzdan birini tanlaydi(sifatini baholash ma'nosida)

3. CallCenter bilan bajariladi
client zayavka qoldiradi(zayavka haqida ma'lumotlarni stateda olgan bo'lamiz buni ham) -> zayavka controllerga keladi ->
-> Controller esa bu zayavkani CallCenter boshlig'iga biriktiradi(CallCenter boshlig'i ro'l xozircha bizda yo'q) -> CallCenter boshlig'i callcaenter operatoriga biriltiradi ->
-> CallCenter esa client bilan qo'ng'iroq qilib bog'lanadi -> CallCenter operatori muammoni kompyuteridan xal qiladi ->
-> CallCenter operatori esa zayakaga qo'shimcha izoh qo'shib zayavkani yopadi ->  bu haqida clientga zayavka haqida ma'lumot yuboriladi ->
-> client esa zayavka to'g'riligini tasdiqaydi va 5a yulduzdan birini tanlaydi(sifatini baholash ma'nosida)
